# Sunny-System
一个自制X64操作系统，随便玩玩

模仿WindowsXP ，借鉴MacOS的一些特效，主要采用C/C++ 语言,并添加一些类似Html javascript css3的标签GUI编程，开发美观有光效特效动画的操作系统，让前端网页的很多控件,网页APP 可以比较容易移植到该操作系统的APP程序。

不走平面化路线，而是大量采用3D图标，2.5D图标，动画图标，光效特效， 半透明，渐变等复杂图形。窗口可以毛玻璃化，像素透明度融合。

暂时不搞文件系统，底层驱动，只要实现X64的多任务调度即可。

图形暂时不启用OPENGL,因为涉及显卡PCI驱动没法实现, 就试试某些底层库CPU渲染。AGG LibSkia Cario 。像素格式统一RGBA8888 RGBA10101010 RGBA16161616 等。
一个窗口并不就是屏幕的一个矩形区，而是可能多个半透明层叠加的不规则图形。

API模仿兼容WindowsXP的，但是适当简化贴图操作，图形混合等开发。

能够用VS2008直接编译出APP程序 跑在这个操作系统上？或者自己开发一个图形APP编译器。

文字采用宋体 边缘锐利，不要那种模糊的雅黑字体， 标配深色浅色等几种风格。

图形GUI的API可以先定版，要先可以展示给用户使用。 至于内核和驱动，文件系统等API慢慢升级完善。

不采用已有的任何工业标准？历史累赘？  类Unix , X Windows ,C语言标准？

可运行在桌面X32 X86 ， Arm64 等架构， 简单小巧方便移植

长远来看系统最终要支持类似OFFICE , 浏览器， 3D游戏。
